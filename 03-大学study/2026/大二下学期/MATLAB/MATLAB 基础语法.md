# 📝 MATLAB 基础语法

## 1. 变量命名与类型
* **命名规则**：
    * 区分大小写（`a` 和 `A` 不同）。
    * 必须以字母开头，后续可跟字母、数字或下划线。
    * 避免使用内置函数名（如 `sin`, `i`, `j`, `pi`）作为变量名。
* **数据类型**：
    * 默认类型为 **double**（双精度浮点数）。
    * 在 [[MATLAB 符号运算]] 中会用到 **sym** 类型。

---

## 2. 矩阵与数组创建 (核心)
MATLAB (Matrix Laboratory) 的一切皆为矩阵。

### 2.1 直接创建
* **行向量**：`row = [1 2 3]` 或 `row = [1, 2, 3]`
* **列向量**：`col = [1; 2; 3]`
* **矩阵**：`A = [1 2; 3 4]`

### 2.2 序列生成
* **冒号运算符**：`x = start:step:end`
    * 示例：`t = 0:0.01:1;` % 生成 0 到 1，步长为 0.01 的向量（常用于 [[信号与系统的 MATLAB 实现]]）
* **等间距线性采样**：`linspace(start, end, n)`
    * 示例：`x = linspace(0, 2*pi, 100);` % 生成 100 个采样点

### 2.3 特殊矩阵
* `zeros(m, n)`：全零矩阵。
* `ones(m, n)`：全一矩阵。
* `eye(n)`：单位矩阵。
* `rand(m, n)`：0~1 均匀分布随机矩阵。

---

## 3. 矩阵索引与切片
> [!CAUTION] 注意
> MATLAB 的索引是从 **1** 开始的，而不是 0。

* `A(m, n)`：访问第 m 行，第 n 列的元素。
* `A(m, :)`：访问第 m 行的所有元素。
* `A(:, n)`：访问第 n 列的所有元素。
* `A(end)`：访问最后一个元素。

---

## 4. 运算规则：矩阵运算 vs 点运算
这是初学者最容易出错的地方，特别是在绘制波形时。

| 运算符 | 矩阵运算 (Matrix) | 点运算 (Element-wise) | 场景说明 |
| :--- | :--- | :--- | :--- |
| **乘法** | `*` | `.*` | 点乘用于对应元素相乘 |
| **除法** | `/` | `./` | 点除常用于信号增益计算 |
| **乘方** | `^` | `.^` | 绘制 $y = t^2$ 必须用 `t.^2` |

> [!EXAMPLE] 为什么需要点运算？
> 如果 `t = [1, 2, 3]`，执行 `t^2` 会报错（因为矩阵不能自乘），必须执行 `t.^2` 得到 `[1, 4, 9]`。

## 5. 常用数学函数
* 三角函数：`sin(x)`, `cos(x)`, `tan(x)`（输入为弧度）。
* 指数/对数：`exp(x)`, `log(x)`（自然对数 $ln$）, `log10(x)`。
* 复数处理：`abs(z)`（取模/幅值）, `angle(z)`（取相角）, `conj(z)`（共轭）。
    * *注：在 [[连续 LTI 系统频域分析]] 中经常用到这些。*
