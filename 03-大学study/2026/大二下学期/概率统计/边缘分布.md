

---

### 1. 计算公式

假设 $(X, Y)$ 的联合概率密度函数为 $f(x, y)$：

- **$X$ 的边缘概率密度函数 $f_X(x)$**：
    
    为了消除 $Y$ 的影响，我们需要在 $Y$ 的整个取值范围内对 $f(x, y)$ 进行积分：
    
    $$f_X(x) = \int_{-\infty}^{+\infty} f(x, y) \, dy$$
    
- **$Y$ 的边缘概率密度函数 $f_Y(y)$**：
    
    同理，为了消除 $X$ 的影响，在 $X$ 的整个取值范围内对 $f(x, y)$ 进行积分：
    
    $$f_Y(y) = \int_{-\infty}^{+\infty} f(x, y) \, dx$$
    

---

### 2. 直观理解：从“面”到“线”

如果把联合概率密度 $f(x, y)$ 想象成底面在 $xOy$ 平面上的一座“土丘”，其高度代表了概率密度的大小：

1. **联合分布**：描述的是点 $(x, y)$ 在平面上落入某个区域的概率，对应的是土丘的**体积**。
    
2. **边缘分布 $f_X(x)$**：相当于你沿着 $y$ 轴的方向把整座土丘“拍扁”投影到 $x$ 轴上。
    
3. **物理意义**：积分 $\int f(x, y) dy$ 实际上是在计算切面面积。对于每一个固定的 $x$，我们将所有可能的 $y$ 产生的概率密度累加起来。
    

---

### 3. 计算步骤与注意事项

在实际解题或应用（如信号处理或数据分析）中，计算边缘分布通常遵循以下步骤：

- **确定积分限**：这是最容易出错的地方。必须根据联合密度函数 $f(x, y)$ 的**非零区域**来确定。如果 $X$ 和 $Y$ 相互独立，积分限通常是常数；如果它们相关（例如在 $0 < y < x < 1$ 区域内有定义），积分限就会包含变量。
    
- **分段讨论**：边缘分布函数通常也是分段函数。在积分完成后，务必标注 $x$ 或 $y$ 的取值范围，在定义域外密度为 0。
    
- **验证**：计算出的边缘分布必须满足概率密度的性质，即 $\int_{-\infty}^{+\infty} f_X(x) dx = 1$。
    

---

### 4. 边缘分布与独立性

边缘分布的一个重要用途是判定变量之间的**独立性**：

> **判定准则**：随机变量 $X$ 和 $Y$ 相互独立的充分必要条件是，对于任意的 $(x, y)$，都有：
> 
> $$f(x, y) = f_X(x) \cdot f_Y(y)$$

如果这个等式不成立，说明 $X$ 和 $Y$ 之间存在统计学上的关联。

---

### 典型例题

假设二维随机变量 $(X, Y)$ 的联合概率密度函数为：

$$f(x, y) =

\begin{cases}

cx, & 0 \le y \le x \le 1 \

0, & \text{其他}

\end{cases}$$

**要求：**

1. 求常数 $c$；
    
2. 求 $X$ 的边缘概率密度 $f_X(x)$；
    
3. 求 $Y$ 的边缘概率密度 $f_Y(y)$。
    

---

### 解题步骤

#### 第一步：确定常数 $c$

利用全概率性质 $\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(x, y) dxdy = 1$。

根据定义域 $0 \le y \le x \le 1$，我们先对 $y$ 积分，再对 $x$ 积分：

$$\int_{0}^{1} dx \int_{0}^{x} cx \, dy = \int_{0}^{1} [cxy]_0^x \, dx = \int_{0}^{1} cx^2 \, dx = \frac{c}{3} [x^3]_0^1 = \frac{c}{3}$$

令 $\frac{c}{3} = 1$，得 **$c = 3$**。

#### 第二步：求 $X$ 的边缘概率密度 $f_X(x)$

求 $X$ 时，要把 $y$ “积掉”。观察定义域 $0 \le y \le x \le 1$，当 $x$ 固定时，$y$ 的取值范围是从 $0$ 到 $x$。

$$f_X(x) = \int_{0}^{x} 3x \, dy = [3xy]_0^x = 3x^2, \quad 0 \le x \le 1$$

**结果：**

$$f_X(x) =

\begin{cases}

3x^2, & 0 \le x \le 1 \

0, & \text{其他}

\end{cases}$$

#### 第三步：求 $Y$ 的边缘概率密度 $f_Y(y)$

求 $Y$ 时，要把 $x$ “积掉”。观察定义域 $0 \le y \le x \le 1$，当 $y$ 固定时，$x$ 的下限是 $y$，上限是 $1$。

$$f_Y(y) = \int_{y}^{1} 3x \, dx = \left[ \frac{3}{2}x^2 \right]_y^1 = \frac{3}{2}(1 - y^2), \quad 0 \le y \le 1$$

**结果：**

$$f_Y(y) =

\begin{cases}

\frac{3}{2}(1 - y^2), & 0 \le y \le 1 \

0, & \text{其他}

\end{cases}$$

---

### 关键点拨

- **画图是灵魂**：在处理这类题目时，一定要先在草稿纸上画出 $0 \le y \le x \le 1$ 这个三角形区域。
    
- **积分限的寻找**：
    
    - 求 $f_X(x)$ 时，作一条**垂直于 $x$ 轴**的射线，看它从哪里进入区域（$y=0$），从哪里离开（$y=x$）。
        
    - 求 $f_Y(y)$ 时，作一条**平行于 $x$ 轴**的射线，看它从哪里进入区域（$x=y$），从哪里离开（$x=1$）。
        

这个例子中，$f_X(x) \cdot f_Y(y) \neq f(x, y)$，所以在这个区域内 $X$ 和 $Y$ 是**不独立**的。
