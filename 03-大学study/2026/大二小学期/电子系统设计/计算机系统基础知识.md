## **CPU(Central Processing Unit)**
- ALU(Arithmetic Logic Unit)算术逻辑单元
	- 执行所有的算术运算及逻辑运算+，-，\*，AND，OR，NOT
	- 运算器所进行的全部操作都是由控制器发出的控制信号来指挥的
	- ![[Pasted image 20260709200055.png]]
- Register（寄存器组）
	- 记录CPU运算的结果；
	- 保存CPU当前的状态；
	- CU根据相应寄存器的值做出决策；
- Control Unit (CU)
	-  “决策机构”，完成协调和指挥整个系统的操作；
	- 从内存中取出一条指令，对指令进行译码，并产生相应
	- 的操作控制信号，以便启动相应的动作；
	- 控制ALU计算，并将结果回写寄存器或者内存；

### 示例代码
```C
int x = 4, y = 1;
unsigned int *p;
while (x > 1) {
y = y * x;
x --;
}
p = 0x100;
*p = y;
```

```汇编伪代码
#1 设置R1 = 4;
#2 设置R2 = 1;
#3 判断R1是否大于1，结果存入
R3寄存器
#4 若R3寄存器为0跳转至#8
#5 计算R2*R1，结果写回R2
#6 计算R1-1，结果写回R1
#7 跳转至#3
#8 设置R4为0x100
#9 结果R2写回R4指向的内存
```

```ASM
#1 SET R1, 4; R1 = 4;
#2 SET R2, 1; R2 = 1;
#3 CMP R1, 1, R3
; IF R1>1 THEN R3=1
#4 JZ R3, 0x08; IF R3==0跳#8
#5 MUL R2, R2, R1; R2 =R2*R1
#6 SUB R1, R1, 1; R1 = R1-1
#7 JMP 0x03; 跳转至#3
#8 SET R4, 0x100; R4=0x100
#9 STORE [R4], R2; *R4=R2
```
