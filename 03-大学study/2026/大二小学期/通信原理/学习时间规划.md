## 🎯 每日学习流程

### 看课日（周一 ~ 周五 + 周六上午）

```
□ 第一步：预习（5 min）
   - 快速浏览教材对应章节的标题、图表、公式
   - 带着问题去看课

□ 第二步：看课（1.5h）
   - 1x 速度听概念，1.5x 速度听例题
   - 关键公式截图或手抄
   - 不懂的地方打标记 ⚠️，不要卡住继续往下

□ 第三步：笔记（30 min）
   - 当天整理 Obsidian 笔记到 [[通信原理/笔记/]]
   - 格式：概念 → 框图 → 公式 → 例题
   - 不懂的地方查资料补充

□ 第四步：习题（可选，周末集中做）
   - 课后习题 2-3 道
   - 错题记录到 [[通信原理/习题/]]
```

### 复习日（周六下午 ~ 周日）

```
□ 周回顾（1h）
   - 复习本周所有笔记
   - 画一张 Mermaid 或 Excalidraw 知识图
   - 更新 [[学习进度追踪]] 中的周日志

□ 习题（1h）
   - 集中完成本周课后习题
   - 标记错题，分析错误原因

□ 弹性（可选）
   - 看其他老师的补充视频
   - 预习下周内容
```

## 💡 避坑提醒

1. **不要在概念上钻牛角尖** — 通信原理很多公式从"模型"出发，先接受再理解
2. **框图比公式更重要** — 弄清楚"信号从哪里来到哪里去"比背公式更有用
3. **数学工具要过关** — 傅里叶变换、概率论是基础，卡住时回去复习 [[信号与系统]] 和 [[概率统计]]
4. **调制方式对比记忆** — 把 AM/FM/ASK/FSK/PSK/QAM 放在一起对比效率/带宽/抗噪性
