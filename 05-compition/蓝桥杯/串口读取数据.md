## 串口接收数据
main.c文件中
``` C
//使能函数
HAL_UART_Receive_IT(huart,(uint8_t*)&_rxBuff,1)

```
因为_rxBuff[1]变量的定义在fun.c中，如果要让主函数识别这个变量，需要在headfile.h写入extern  \_rxBuff 
>[!tip] 另一种写法


在回调函数中，只写一句
- HAL_UART_Receive_IT(huart,(uint8_t \*)&\_rxBuff,1);
- 可能无法处理多数据情况？


fun.c文件中
```C
//存储串口1接受的数据
uint8_t Rxbuff[USARTMAXLENTH],_rxBuff[1];
//记录串口接受到的数据大小
uint16_t RxCount = 0;
//使用中断之后的回调函数，每次接收数据完成后执行该函数
void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart)
{
	if(haurt->instance == USART1)//此处代码与之后的计时器回调类似
	{
	//加上下面这两句可以是的串口数据接收变为接收长数据？
	Rxbuff[RxCount++] = _rxBuff[0];
	RxCount %= 7;
	//
	HAL_UART_Receive_IT(huart,(uint8_t *)&_rxBuff,1);
	}
}
```
- 因为在使能函数中调用的是指针形式的_rxBuff，所以定义中有_rxBuff[1]的写法
## 串口发送数据
- 使用函数有： 
- void HAL_UART_TxCpltCallback (UART_HandleTypeDef \*huart) 中断发送函数
- 更推荐使用使用
- 函数HAL_StatusTypeDef  HAL_UART_Transmit(UART_HandleTypeDef \*huart, const uint8_t 、\*pData, uint16_t Size, uint32_t  Timeout)
 ### 函数解析
 ```C
 HAL_StatusTypeDef HAL_UART_Transmit(UART_HandleTypeDef *huart, const uint8_t  *pData, uint16_t Size, uint32_t Timeout) 
/* UART_HandleTypeDef *huart:串口通道 const uint8_t *pData:发送的数据 uint16_t Size:发送数据的大小  uint32_t Timeout:发送数据超时时间 */ 
/* 使用示例：  发送一个字符串"hi",其超时时间为50ms */  
 HAL_UART_Transmit(&huart1,(unsigned char*)"hi",sizeof("hi"),50)
```
# 使用DMA

- 初始化
```
HAL_UART_Receive_DMA(&huart1,_ucRxbuff,1);

```

```C
//定义一个串口信息的结构 
uint8_t ucRxbuff[10]; 
uint8_t _ucRxbuff[1],lenBuff = 0; 
extern DMA_HandleTypeDef hdma_usart1_rx; 
extern DMA_HandleTypeDef hdma_usart1_tx; 
/***使用HAL_UART_Receive_IT中断接收数据 每次接收完成数据后就会执行该函数***/ void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart) {  
 if(huart->Instance == USART1){  ucRxbuff[lenBuff++%10] = _ucRxbuff[0];  } 
}

```

每次收到一个字符就将其放置到buff中，等待处理，一旦数据处理完成后需要将buff清空。
使用示例：
```C
// 避免重复处理空buff 此处需要判断buff的长度 if(strlen((char*)ucRxbuff) != 0 ) { 
    LCD_ClearLine(Line2);    
	LCD_DisplayStringLine(Line2,(u8*)ucRxbuff);         
	memset(ucRxbuff,0,10); 
    lenBuff = 0; 
    }
```
