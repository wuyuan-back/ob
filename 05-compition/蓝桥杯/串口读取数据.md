## 串口发送数据
- 使用函数有： 
- void HAL_UART_TxCpltCallback (UART_HandleTypeDef \*huart) 中断发送函数
- 更推荐使用使用
- 函数HAL_StatusTypeDef  HAL_UART_Transmit(UART_HandleTypeDef \*huart, const uint8_t 、\*pData, uint16_t Size, uint32_t  Timeout)
 ### 函数解析
 ```C
 HAL_StatusTypeDef HAL_UART_Transmit(UART_HandleTypeDef *huart, const uint8_t  *pData, uint16_t Size, uint32_t Timeout) 
/* UART_HandleTypeDef *huart:串口通道 const uint8_t *pData:发送的数据 uint16_t Size:发送数据的大小  uint32_t Timeout:发送数据超时时间 */ 
/* 使用示例：  发送一个字符串"hi",其超时时间为50ms */  
 HAL_UART_Transmit(&huart1,(unsigned char*)"hi",sizeof("hi"),50)
```

# 利用定时器进行串口的不定时长字符接收
题目重现：12届省赛题
串口接收：每次进入中断，只能接收一个字节的数据
- 考虑错误情况：
	- 单个，方法1，不采用
	- 总体判断，判断字符串，
## 如何判断接受完所有字符
- 串口传输一次10bit，波特率9600，1s传9600bit,10\*1/9600,所以传输一次的时间约为1.04ms
- 意味着只要串口在1.04ms后没有收到数据说明传输结束
- 这里考虑使用[[定时器中断]]的方法
- PSC+1 = 8000，8000/80$*10^6$ = 1/10000 = 0.1ms
- 当CNT = 15时 t>1.5ms >1.04所以可以判断
## 代码实现
- 在串口中断中，添加标志位rec_flag,用于判断此时是否接收数据
