# 不使用DMA
- CubeMX配置
	- usart1串口默认配置PC4,PC5->修改为PA9,PA10
	- 在左边的搜索栏中查找usart1,Mode调为Asynchrous
## 串口接收数据
main.c文件中
``` C
//使能函数
HAL_UART_Receive_IT(huart,(uint8_t*)&_rxBuff,1)

```
fun.c文件中
```C
//存储串口1接受的数据
uint8_t Rxbuff[USARTMAXLENTH],_rxBuff[1];
//记录串口接受到的数据大小
uint16_t RxCount = 0;
//使用中断之后的回调函数，每次接收数据完成后执行该函数
void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart)
{
	if(haurt->instance == USART1)//此处代码与之后的计时器回调类似
	{
	//加上下面这两句可以是的串口数据接收变为接收长数据？
	Rxbuff[RxCount++] = _rxBuff[0];
	RxCount %= 7;
	//
	HAL_UART_Receive_IT(huart,(uint8_t *)&_rxBuff,1);
	}
}
```
- 因为在使能函数中调用的是指针形式的_rxBuff，所以定义中有_rxBuff[1]的写法
