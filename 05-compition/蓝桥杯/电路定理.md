## 1. 核心理论：等效变换与功率匹配

在含有独立源的线性电阻网络中，二端网络对外电路的效应可以用一个理想电源模型替代。

### A. 戴维南定理 (Thevenin's Theorem)

- **定义**：等效为一个电压源 $U_{oc}$ 串联一个电阻 $R_{th}$。
- **本质**：电压响应=Uoc(开路电压) +Req\* I
- **求解等效电阻**：
	- 外加电压法，外加电流(模电常用)
	- 开路电压，短路电流法
- **最大功率传输**：当负载 $R_L = R_{th}$ 时，$P_{max} = \frac{U_{oc}^2}{4R_{th}}$。
    

> [!IMPORTANT] 物理意义
> 
> 该定理研究的是**阻抗匹配**。在信号传输中，为了获得最大输出，必须使负载电阻等于电源内阻。

## 2. 重点案例实战 (Case Studies)

### 案例 1：最大功率传输与电阻化简

**【例题】** 求负载 $R_L$ 为多少时可获得最大功率？
![[mmexport1774605128843_edit_49614791489769.jpg]]
**解析步骤：**

1. **独立源置零**：电压源 $\to$ 短路；电流源 $\to$ 断路。
    
2. **结构化电阻化简**：
    
    - $R_1 = (6 \parallel 12) = 4\Omega$
        
    - $R_2 = 4 + 3 = 7\Omega$
        
    - $R_{th} = 7 + 2 = 9\Omega$
        
3. **结论**：当 $R_L = 9\Omega$ 时，获得最大功率。
    

---

### 案例 2：含受控源的短路电流计算

**【例题】** 计算 $a, b$ 端口的短路电流 $I_{sc}$。
![[mmexport1774605414008_edit_49597588688109.jpg]]

**解析步骤：**

1. **锁定控制量 $i_x$**：短路后，$4\Omega$ 电阻电压被 $10\text{V}$ 电源锁定，故 $i_x = 10/4 = 2.5\text{A}$。
    
2. **受控源输出**：$2i_x = 5\text{A}$。
    
3. **KCL 节点方程**：
    
    - 电阻支路电流 $I_1 = 10/5 = 2\text{A}$。
        
    - 总短路电流 $I_{sc} = I_1 + 2i_x = 2 + 5 = 7\text{A}$。
        

> [!WARNING] 常见易错点
> 
> **受控源不可置零**！它随电路状态变化，必须保留在方程中。

### 案例 3：叠加定理求解开路电压

**【例题】** 求 $a, b$ 端口的开路电压 $U_{oc}$。

![[mmexport1774605692682_edit_49575202046711.jpg]]

**解析步骤：**

1. **$32\text{V}$ 单独作用**：分压得到 $U_1 = 24\text{V}$。
    
2. **$2\text{A}$ 单独作用**：并联电压 $U_2 = 2 \times (4 \parallel 12) = 6\text{V}$。
    
3. **叠加**：$U_{oc} = 24 + 6 = 30\text{V}$。

## 3. 重点与解题陷阱 (Checklist)

> [!TIP] 记忆窍门
> 
> **“压短流断”**：求 $R_{th}$ 时，电压源看作导线，电流源看作开路。

- [ ] **开路端串联电阻**：求 $U_{oc}$ 时，若电阻一端悬空，其电压降必为 0。
    
- [ ] **受控源极性**：严格按照题目给定的 $i_x$ 或 $u_x$ 方向列 KCL/KVL 方程。
    
- [ ] **电源变换法**：当叠加法计算繁琐时，考虑将“电压源串电阻”改为“电流源并电阻”。

# ⚡ 戴维南等效电阻 ($R_{th}$) 求解全攻略

求解 $R_{th}$ 的本质是计算从端口看进去的“入端电阻”。根据电路中是否含有**受控源**，选择的方法也有所不同 。

---

## 方法一：独立源置零法 (网络化简法)

**适用场景**：电路中**仅含有独立源**和电阻，==不含受控源== 。

- **操作步骤**：
    
    1. 将电路内所有独立源置零：电压源 $\to$ **短路**；电流源 $\to$ **断路** 。
        
    2. 利用电阻的串、并联公式直接计算端口 $a, b$ 间的等效电阻 。
        
- **例题结合 (对应题目 8/10)**：
    
    - **置零**：左侧 $12\text{V}$ 电源变导线，中间 $2\text{A}$ 电源变开路 。
        
    - **计算**：$R_{th} = (6\Omega \parallel 12\Omega) + 3\Omega + 2\Omega = 4 + 3 + 2 = 9\Omega$。
        

---

## 方法二：开路电压/短路电流法 ($U_{oc} / I_{sc}$)

**适用场景**：通用方法，尤其适用于含有**受控源**的复杂电路 。

- **操作步骤**：
    
    1. 计算端口的开路电压 $U_{oc}$ 。
        
    2. 将端口短路，计算流过短路支路的电流 $I_{sc}$ 。
        
    3. 利用欧姆定律求解：$R_{th} = \frac{U_{oc}}{I_{sc}}$ 。
        
- **例题结合 (综合题目 9/10 与 10/10 的逻辑)**：
    
    - 若已知某电路 $U_{oc} = 30\text{V}$（如题 10），且测得短路电流 $I_{sc} = 10\text{A}$。
        
    - 则该电路的戴维南等效电阻 $R_{th} = \frac{30\text{V}}{10\text{A}} = 3\Omega$。
        

---

## 方法三：外加电源法 (加压求流法)

**适用场景**：当电路中含有**受控源**，且内部没有独立源（或==独立源已置零==）时 。

- **操作步骤**：
    
    1. 将电路内所有**独立源置零**（受控源保留）。
        
    2. 在端口 $a, b$ 外加一个已知电压源 $U_{test}$（通常设为 $1\text{V}$）或已知电流源 $I_{test}$（通常设为 $1\text{A}$）。
        
    3. 计算端口流出的电流 $I$（或两端电压 $U$）。
        
    4. $R_{th} = \frac{U_{test}}{I}$ 或 $R_{th} = \frac{U}{I_{test}}$ 。
        

> [!TIP] 为什么不能直接串并联？ 因为受控源的数值随电路状态改变。外加电源法能通过 $U/I$ 的比值，将受控源对电路阻抗的“放大”或“缩小”效应准确反映出来 。

## 📝 综合例题演练

**【题目】** 求下图电路的 $R_{th}$。已知内部含有一个受控电流源 $2i_x$，控制量 $i_x$ 为某支路电流。

1. **第一步：置零**。将所有独立电压源短路，独立电流源断路 。
    
2. **第二步：加压**。在端口施加 $U_{test} = 1\text{V}$。
    
3. **第三步：列方程**。根据 KCL/KVL 找出 $I$ 与 $U_{test}$ 的关系。假设算得 $I = 0.5\text{A}$。
    
4. **第四步：结果**。$R_{th} = 1\text{V} / 0.5\text{A} = 2\Omega$。
    

---

## 🧠 避坑 Checklist

- [ ] **独立源 vs 受控源**：置零法只针对独立源 。
    
- [ ] **功率不可叠加**：即使求出了各电源单独作用的电流，计算总功率也必须先叠加电流再用 $I^2R_{th}$，不可直接叠加功率 。
    
- [ ] **开路电阻**：求 $U_{oc}$ 时，与端口串联的电阻不分压 。
