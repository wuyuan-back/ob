[[examle]]### 方式二：结构化描述（Vibe Coding 方式）

帮我做一个计数器组件，要求：
- 目标：一个带加减按钮的数字计数器
- 约束：使用 HTML + CSS + 原生 JavaScript，单文件
- 输入：初始值 0，最小值 0（不能减到负数），最大值 99
- 输出：显示当前数字，两个按钮（+ 和 -），一个重置按钮
- 验收标准：
  - 点 + 数字加 1，到 99 后按钮变灰不可点击
  - 点 - 数字减 1，到 0 后按钮变灰不可点击
  - 点重置回到 0
  - 数字变化时有短暂的放大动画
  - 深色背景，白色卡片，圆角设计
