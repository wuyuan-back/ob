| 要素   | 核心问题    | 如果缺失会怎样               |
| ---- | ------- | --------------------- |
| 目标   | 要做什么？   | AI 猜你的意图，可能偏离方向       |
| 约束   | 有什么限制？  | AI 用你不熟悉的技术栈，代码无法集成   |
| 输入   | 接收什么数据？ | AI 假设输入格式，实际运行时出错     |
| 输出   | 产生什么结果？ | AI 给了代码但没给使用方式，不知道怎么跑 |
| 验收标准 | 怎么算合格？  | AI 做完了但你不知道算不算做完      |
