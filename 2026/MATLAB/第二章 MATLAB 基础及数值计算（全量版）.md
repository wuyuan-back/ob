# 🔢 第二章 MATLAB 基础及数值计算（全量版）

## 1. 基本概念 (Basic Concepts)

### 1.1 变量与命名
* **命名规则**：
    * 必须以**字母**开头。
    * 由字母、数字、下划线组成。
    * **区分大小写**（例如：`pi` 与 `PI` 是不同的变量）。
    * 长度上限为 **63 个字符**。
* **特性**：无需预先声明类型，MATLAB 会根据赋值自动分配存储空间。

### 1.2 预定义变量 (Predefined Variables)
| 变量 | 含义 | 备注 |
| :--- | :--- | :--- |
| **ans** | 默认结果变量 | 存储最近一次未赋值的运算结果 |
| **eps** | 最小机器数 | 约 $2.22 \times 10^{-16}$，常用于避免除零错误 |
| **inf / Inf** | 无穷大 | 如 $1/0$ 的结果 |
| **NaN / nan** | 非数值 | Not a Number，如 $0/0$ 或 $\infty/\infty$ |
| **i / j** | 虚数单位 | $i^2 = j^2 = -1$ |
| **pi** | 圆周率 $\pi$ | $3.14159...$ |
| **realmax** | 最大正实数 | 约 $1.79 \times 10^{308}$ |
| **realmin** | 最小正实数 | 约 $2.22 \times 10^{-308}$ |

### 1.3 数值表示
* **存储精度**：默认以双精度（double）存储。
* **显示格式**：使用 `format` 命令改变显示（不影响实际计算精度）。
    * `format short`：4位小数（默认）。
    * `format long`：15位小数。
    * `format short e`：科学计数法。
* **复数表示**：`a = 1 + 2i` 或 `a = 1 + 2*j`。

### 1.4 MATLAB 语句规范
* **形式**：`表达式` 或 `变量 = 表达式`。
* **结尾符号**：
    * 分号 `;`：执行语句但不显示结果。
    * 逗号 `,`：在一行内分隔多条语句并显示结果。
* **示例**：`>> (12+2*(7-4))/3^2` $\rightarrow$ `ans = 2`。

## 2. 矩阵：核心数据结构

### 2.1 创建矩阵
* **直接输入法**：使用方括号 `[]`，行内元素用**空格或逗号**分隔，行间用**分号** `;` 分隔。
    * `a = [1, 2; 3, 4]`
    * `b = [2, 1+3j; sqrt(4), 5]`
* **常用函数创建**：
    * `[]`：空矩阵。
    * `ones(N, M)`：全 1 矩阵。
    * `zeros(N, M)`：全 0 矩阵。
    * `rand(N, M)`：$[0, 1]$ 均匀分布随机矩阵。
    * `randn(N, M)`：标准正态分布随机矩阵。
    * `eye(N)`：$N \times N$ 单位矩阵。

### 2.2 向量生成
* **冒号运算符**：`a = m : step : n`
    * 示例：`tt = -4*pi : pi/100 : 4*pi;`（信号采样常用，产生 801 个点）。
* **linspace 函数**：`linspace(m, n, k)`
    * 生成从 $m$ 到 $n$ 之间均匀分布的 $k$ 个点。

### 2.3 矩阵引用的进阶技巧
* **元素访问**：`A(i, j)` 指向第 $i$ 行第 $j$ 列（**注意：下标从 1 开始**）。
* **子块引用**：`A(:, [1, 3])` 表示提取所有行，但只取第 1 和第 3 列。
* **矩阵拼接**：
    * 水平：`C = [A, B]`
    * 垂直：`C = [A; B]`

### 2.4 矩阵信息与变换
* **基本信息**：`size(A)` (大小), `length(A)` (向量长度), `ndims(A)` (维数), `isempty(A)` (判空)。
* **矩阵变换**：
    * `fliplr(A)`：左右翻转。
    * `flipud(A)`：上下翻转。
    * `rot90(A)`：逆时针旋转 90°。

## 3. 矩阵运算 (Matrix Operations)

### 3.1 线性代数运算
| 运算 | 符号 | 说明 |
| :--- | :--- | :--- |
| **加减** | `+ -` | 必须同维，或其中一个是标量 |
| **乘法** | `*` | 满足矩阵乘法规则（左列=右行） |
| **转置** | `'` | **共轭转置**（复数矩阵会取共轭） |
| **幂次** | `^` | 要求必须是方阵，如 `A^2` 等于 `A*A` |
| **左除** | `\` | 解方程 $AX = B \Rightarrow X = A \backslash B$ (常用) |
| **右除** | `/` | 解方程 $XA = B \Rightarrow X = B / A$ |
| **其他** | `inv`, `det`, `eig`, `poly` | 求逆、行列式、特征值、特征多项式 |

### 3.2 数组运算（逐元素逐点运算）
> [!IMPORTANT] 信号处理核心点
> 在计算信号波形（如 $y = \sin(t) \cdot e^{-t}$）时，必须使用**点运算**。

* `.*`：对应元素相乘。
* `./` 或 `.\`：对应元素相除。
* `.^`：逐元素幂次。
* **示例**：`a = [1,2,3]; b = [4,5,6];`
    * `c = a .* b` $\rightarrow$ `[4, 10, 18]`
    * `d = a .^ 2` $\rightarrow$ `[1, 4, 9]`

## 4. 关系与逻辑运算
* **关系符**：`>`, `<`, `>=`, `<=`, == (等于), `~=` (不等于)。
* **逻辑符**：`&` (与), `|` (或), `~` (非)。
* **规则**：比较结果为逻辑值（1 或 0）。同维矩阵对应元素比较，或标量与矩阵每个元素比较。

## 6. 多项式 (Polynomials)
在 MATLAB 中，多项式用系数向量表示：$P(s) = a_n s^n + a_{n-1}s^{n-1} + \dots + a_0$ 记为 `P = [a_n, a_{n-1}, ..., a_0]`。

* **创建**：`A = [1 2 0 1]` 表示 $s^3 + 2s^2 + 1$。
* **求根**：`roots(P)`。
* **还原**：`poly(r)`。
* **相乘/卷积**：`conv(A, B)`。
* **相除(带余除法)**：`[Q, r] = deconv(R, A)`。
* **微分**：`polyder(P)`。

## 💡 综合示例：Sa(t) 信号处理
**解决除零问题的经典方案：**
```matlab
tt = -4*pi : pi/100 : 4*pi;
w = 1;
% (tt==0) 在零点处返回 1，其余为 0
% t 将 0 替换为微小量 eps，避免计算 y 时分母为 0
t = (tt==0)*eps + tt; 
y = w * sin(w*t) ./ (pi * w * t);
plot(t, y);
```

## ✍️ 作业与易错点总结
1.  **3+4i 与 3+4*i**：前者是常数，后者涉及变量 `i` 的查找。若 `i` 被定义（如 `for i=1:10`），后者会出错。
2.  **linspace 转换**：`linspace(1, 10, 11)` 步长为 $(10-1)/(11-1)=0.9$，等价于 `1:0.9:10`。
3.  **除法区别**：`A/B` 是 $A \times B^{-1}$，`B\A` 是 $B^{-1} \times A$。
4.  **运算模式选择**：绘制函数曲线必选点运算（`.*`, `./`, `.^`）。
