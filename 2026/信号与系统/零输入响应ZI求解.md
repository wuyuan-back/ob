# 📝 信号与系统：零输入响应 (Zero-Input Response, ZI)

## 1. 定义与物理意义

**零输入响应 $r_{zi}(t)$** 指的是：当系统的输入激励 $e(t) = 0$ 时，仅由系统的初始储能（初始状态）引起的响应。

- **核心逻辑**：系统“吃老本”，靠电容电压或电感电流维持的后续反应。
    
- **数学本质**：齐次微分方程在给定初始条件下的**特解**（在数学上它是通解形式，但系数由初始值确定）。
    

---

## 2. 标准求解流程

### 步骤一：建立齐次微分方程

令系统的激励 $e(t) = 0$ 及其各阶导数为 0，得到：

$$a_n \frac{d^n}{dt^n}r(t) + a_{n-1} \frac{d^{n-1}}{dt^{n-1}}r(t) + \dots + a_1 \frac{d}{dt}r(t) + a_0 r(t) = 0$$

### 步骤二：求特征根

写出特征方程：

$$a_n \lambda^n + a_{n-1} \lambda^{n-1} + \dots + a_1 \lambda + a_0 = 0$$

解出特征根 $\lambda_1, \lambda_2, \dots, \lambda_n$。

### 步骤三：写出齐次解形式 (即 $r_{zi}(t)$ 的通用形式)

根据特征根的情况，写出 $r_{zi}(t)$：

1. **单实根**：$C e^{\lambda t}$
    
2. **重根 ($\lambda$ 为 $k$ 重根)**：$(C_1 + C_2 t + \dots + C_k t^{k-1})e^{\lambda t}$
    
3. **共轭复根 ($\lambda = \alpha \pm j\beta$)**：$e^{\alpha t} [A \cos(\beta t) + B \sin(\beta t)]$
    

### 步骤四：代入初始条件确定系数

> [!IMPORTANT] 关键细节：$0_-$ 还是 $0_+$？
> 
> 对于零输入响应，由于没有激励（$e(t)=0$），系统状态**不会发生跳变**。因此：
> 
> $$r_{zi}(0_+) = r_{zi}(0_-) \quad \text{且} \quad r'_{zi}(0_+) = r'_{zi}(0_-)$$
> 
> 直接代入题目给出的 $r(0_-)$ 即可求出待定系数 $C_1, C_2 \dots$。

---

## 3. 经典案例演示

**题目**：已知方程 $r''(t) + 3r'(t) + 2r(t) = e'(t) + 3e(t)$，初始状态 $r(0_-)=1, r'(0_-)=2$，求 $r_{zi}(t)$。

1. **齐次化**：$r''(t) + 3r'(t) + 2r(t) = 0$
    
2. **特征根**：$\lambda^2 + 3\lambda + 2 = 0 \Rightarrow \lambda_1 = -1, \lambda_2 = -2$
    
3. **写形式**：$r_{zi}(t) = C_1 e^{-t} + C_2 e^{-2t}$
    
4. **定系数**：
    
    - $r_{zi}(0) = C_1 + C_2 = 1$
        
    - $r'_{zi}(0) = -C_1 - 2C_2 = 2$
        
    - 解方程组：$C_1 = 4, C_2 = -3$
        
5. **结论**：$r_{zi}(t) = (4e^{-t} - 3e^{-2t})u(t)$
    

---

## 4. 常见误区分析

> [!CAUTION] 易错点
> 
> 1. **不要看右边**：求零输入响应时，方程右边的所有 $e(t)$ 项统统当做 0。右边的结构（如 $e'(t)$）只影响“零状态响应”。
>     
> 2. **初始值的使用**：零输入响应是唯一可以直接放心使用 $0_-$ 初始值的情况。
>     
> 3. **稳定性判断**：如果特征根 $\lambda$ 的实部大于 0（在右半平面），则 $r_{zi}(t)$ 会随时间无限增长，说明系统不稳定。
>
