# 📔一. 信号与系统单元：算子法 (Simplified Method) 求解单位响应

## 1. 核心思想：系统的“基因”解 $y_n(t)$

算子法的精髓在于：**先求出一个不带任何激励项导数影响的基础响应 $y_n(t)$，然后再把激励项的微分算子作用上去。**

这种方法最大的优势是：**规避了复杂的 $0_-$ 到 $0_+$ 的跳变匹配**，将微分方程的求解变成了纯粹的代数和求导运算。

---

## 2. 标准求解流程 (以 $N$ 阶系统为例)

假设微分方程为：$Q(D)r(t) = P(D)e(t)$

其中 $Q(D)$ 是左端算子，$P(D)$ 是右端算子。

### 第一步：求基础特征模态 $y_n(t)$

1. **齐次化**：令 $Q(D)y_n(t) = 0$。
    
2. **求特征根**：解特征方程 $Q(\lambda) = 0$。
    
3. **确定 $y_n(t)$ 的通用形式**（如 $\sum C_i e^{\lambda_i t}$）。
    
4. **设置“算子初值条件”**：
    
    这是该方法的灵魂。令前 $N-1$ 阶导数为 0，第 $N-1$ 阶导数为 1：
    
    $$y_n(0) = y_n'(0) = \dots = y_n^{(N-2)}(0) = 0, \quad y_n^{(N-1)}(0) = 1$$
    
    > [!NOTE] 物理直觉
    > 
    > 这相当于给系统最高阶导数一个“单位冲激”，使其产生一个单位跳变。
    

### 第二步：合成单位冲激响应 $h(t)$

利用线性时不变性，将右端算子 $P(D)$ 作用于 $y_n(t)$：

$$h(t) = b_0 \delta(t) + [P(D)y_n(t)]u(t)$$

- **$b_0$ 的计算**：仅当 $P(D)$ 与 $Q(D)$ 阶次相等时存在，$b_0 = \text{最高阶系数之比}$。
    

### 第三步：合成单位阶跃响应 $g(t)$

$$g(t) = [P(D) \int_0^t y_n(\tau) d\tau] u(t)$$

或者更简单地：求出 $h(t)$ 后直接积分。

---

## 3. 经典案例演示 (参考手稿 Example 2.6)

**题目**：$(D^2 + 3D + 2)r(t) = De(t)$，求 $h(t)$。

1. **特征根**：$\lambda^2 + 3\lambda + 2 = 0 \Rightarrow \lambda_1 = -1, \lambda_2 = -2$。
    
2. **求 $y_n(t)$**：
    
    - 形式：$y_n(t) = C_1 e^{-t} + C_2 e^{-2t}$。
        
    - 初值：$y_n(0) = 0 \Rightarrow C_1 + C_2 = 0$。
        
    - 初值：$y_n'(0) = 1 \Rightarrow -C_1 - 2C_2 = 1$。
        
    - 解得：$C_1 = 1, C_2 = -1 \Rightarrow y_n(t) = e^{-t} - e^{-2t}$。
        
3. **求 $h(t)$**：
    
    - 右端算子 $P(D) = D$。
        
    - $h(t) = D y_n(t) = \frac{d}{dt}(e^{-t} - e^{-2t}) = (-e^{-t} + 2e^{-2t})u(t)$。
        

---

## 4. 算子法的阶数规律总结

|**情况**|**右端算子 P(D) 阶数 m**|**h(t) 的组成分量**|
|---|---|---|
|**宽泛系统**|$m < n$|仅含 $e^{\lambda t}$ 模式的指数项|
|**临界系统**|$m = n$|包含 $\delta(t)$ 项 + 指数项|
|**高阶微分系统**|$m > n$|包含 $\delta'(t)$ 甚至更高阶奇异函数|

---

## 5. 💡 考试避坑 Tips

- **$u(t)$ 的位置**：在使用 $P(D)y_n(t)$ 时，导数要在 $u(t)$ **之外**求，或者求完导后再整体乘以 $u(t)$。不要把 $u(t)$ 卷进求导过程产生多余的 $\delta$。
    
- **$y_n(t)$ 永远是零状态下的响应**：虽然我们给它设了 $y_n'(0)=1$，但那只是为了数学计算方便，它本质上还是一个零状态分量。
    
- **三阶及以上系统**：初值条件顺延，例如三阶系统就是 $y_n(0)=0, y_n'(0)=0, y_n''(0)=1$。
# 二.算子法证明
要理解算子法（Simplified Method）的推导，核心在于理解**奇异函数匹配**以及**线性时不变系统（LTI）的微分性质**。

我们可以分三步来拆解这个“魔术”背后的原理。

---

### 1. 核心思想：先解决“最简单”的情况

假设系统的微分方程为：

$$Q(D)r(t) = P(D)e(t)$$

其中 $Q(D) = \sum_{a=0}^{N} a_n D^n$，$P(D) = \sum_{b=0}^{M} b_m D^m$。

由于系统是 **线性 (Linear)** 且 **时不变 (Time-Invariant)** 的，我们可以把复杂的激励项 $P(D)e(t)$ 拆开看。如果我们能先求出一个**基础响应 $y_n(t)$**，它满足：

$$Q(D)y_n(t) = e(t)$$

那么，原方程的解 $r(t)$ 就可以简单地表示为：

$$r(t) = P(D)y_n(t)$$

这正是算子法的灵魂：**先求“裸系统”对输入信号的响应，再把输入信号的导数加权叠加上去。**

---

### 2. 为什么初始条件是 $y_n^{(N-1)}(0)=1$？

当你求**单位冲激响应 $h(t)$** 时，基础方程变为：

$$Q(D)y_n(t) = \delta(t)$$

即：$a_N y_n^{(N)}(t) + a_{N-1} y_n^{(N-1)}(t) + \dots + a_0 y_n(t) = \delta(t)$。

为了平衡等式两边的奇异信号（$\delta$ 函数）：

- 等式右边有一个 $\delta(t)$。
    
- 等式左边最高阶项是 $a_N y_n^{(N)}(t)$。
    
- 要想让左边产生一个 $\delta(t)$，**必须且只能**让次高阶项 $y_n^{(N-1)}(t)$ 在 $t=0$ 时刻发生一个幅度为 $1/a_N$ 的**跳变**（从 0 跳到 $1/a_N$）。
    
- 只要 $y_n^{(N-1)}(t)$ 发生了跳变，它的导数 $y_n^{(N)}(t)$ 就会产生一个 $\delta(t)$，从而与右边抵消。
    

因此，为了数学计算方便（设 $a_N=1$），我们规定 $y_n(t)$ 的 $0_+$ 初始条件为：

$$y_n(0) = y_n'(0) = \dots = y_n^{(N-2)}(0) = 0, \quad y_n^{(N-1)}(0) = 1$$

---

### 3. $h(t) = b_0 \delta(t) + [P(D)y_n(t)]u(t)$ 的推导

现在我们有了基础响应 $y_n(t)u(t)$，我们要把它代回原方程的右端项算子 $P(D)$。

根据复合函数求导法则（Leibniz 法则）：

$$\frac{d}{dt} [f(t)u(t)] = f'(t)u(t) + f(0)\delta(t)$$

当你把 $P(D)$ 作用于 $y_n(t)u(t)$ 时：

1. **如果 $M < N$**：由于 $y_n(0), y_n'(0) \dots$ 直到 $N-2$ 阶导数都是 0，所以求导过程中产生的 $\delta$ 项都会因为系数为 0 而消失。最终只剩下 $P(D)y_n(t)u(t)$。
    
2. **如果 $M = N$**：当你求到第 $N$ 阶导数时，由于 $y_n^{(N-1)}(0) = 1$，求导会产生一个 $1 \cdot \delta(t)$。再乘上 $P(D)$ 最高阶的系数 $b_N$，就得到了 $b_0 \delta(t)$ 项（其中 $b_0 = b_N/a_N$）。
    

---

### 4. 总结推导逻辑

1. **分解**：将系统响应拆分为“特征模态”和“激励算子”。
    
2. **匹配**：利用 $\delta(t)$ 强迫系统最高阶导数产生跳变，确定 $y_n(t)$ 的初值。
    
3. **叠加**：利用微分性质，将 $P(D)$ 作用于 $y_n(t)$。在这个过程中，低阶导数的连续性保证了不产生 $\delta$，只有最高阶匹配才会产生 $\delta$。

### 📝 放在 Obsidian 笔记里的“公式版”推导：

> [!ABSTRACT] 算子法推导全过程
> 
> 1. 设 $Q(D)y_n(t) = \delta(t)$
>     
> 2. 积分一次得：$\int_{0_-}^{0_+} [y_n^{(N)} + \dots] dt = \int_{0_-}^{0_+} \delta(t) dt = 1$
>     
> 3. 得跳变值：$y_n^{(N-1)}(0_+) - y_n^{(N-1)}(0_-) = 1$
>     
> 4. 原系统响应：$h(t) = P(D) y_n(t)u(t)$
>     
> 5. 展开：$h(t) = [P(D)y_n(t)]u(t) + \sum_{k=0}^{M-1} P_{k+1} y_n^{(k)}(0)\delta(t) \dots$
>     
> 6. 简化：因 $y_n^{(0 \dots N-2)}(0)=0$，仅当 $M=N$ 时剩下一项 $b_N \delta(t)$。
>
