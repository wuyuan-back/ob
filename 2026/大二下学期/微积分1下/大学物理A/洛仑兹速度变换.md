推导洛伦兹速度变换公式，核心在于利用**洛伦兹坐标变换**对时间求导。

假设有两个惯性系：$S$ 系和 $S'$ 系。$S'$ 系相对于 $S$ 系以恒定速度 $v$ 沿 $x$ 轴正方向运动。一个质点在空间中运动，其在两个参考系中的坐标和时间关系由洛伦兹变换给出。

---

### 1. 准备工作：坐标微分

首先写出从 $S'$ 系到 $S$ 系的**逆洛伦兹变换**（这样推导出的 $u$ 是关于 $u'$ 的函数，更常用）：

1. $dx = \gamma (dx' + v dt')$
    
2. $dy = dy'$
    
3. $dz = dz'$
    
4. $dt = \gamma (dt' + \frac{v}{c^2} dx')$
    

其中 $\gamma = \frac{1}{\sqrt{1-v^2/c^2}}$。

---

### 2. $x$ 方向速度分量的推导

$S$ 系中测得的速度分量 $u_x = \frac{dx}{dt}$。我们将上面的微分式相除：

$$u_x = \frac{\gamma (dx' + v dt')}{\gamma (dt' + \frac{v}{c^2} dx')}$$

分子分母同时约去 $\gamma$，并同时除以 $dt'$：

$$u_x = \frac{\frac{dx'}{dt'} + v}{1 + \frac{v}{c^2} \frac{dx'}{dt'}}$$

因为 $u'_x = \frac{dx'}{dt'}$ 是质点在 $S'$ 系中的速度，所以得到：

> $$u_x = \frac{u'_x + v}{1 + \frac{u'_x v}{c^2}}$$

---

### 3. $y$ 和 $z$ 方向速度分量的推导

注意，虽然洛伦兹变换中 $dy = dy'$，但由于时间 $dt$ 发生了变换，垂直方向的速度**依然会改变**。以 $u_y$ 为例：

$$u_y = \frac{dy}{dt} = \frac{dy'}{\gamma (dt' + \frac{v}{c^2} dx')}$$

同样，分子分母同时除以 $dt'$：

$$u_y = \frac{\frac{dy'}{dt'}}{\gamma (1 + \frac{v}{c^2} \frac{dx'}{dt'})}$$

得到：

> $$u_y = \frac{u'_y}{\gamma (1 + \frac{u'_x v}{c^2})} = \frac{u'_y \sqrt{1 - \frac{v^2}{c^2}}}{1 + \frac{u'_x v}{c^2}}$$

同理可得 $z$ 方向：

> $$u_z = \frac{u'_z \sqrt{1 - \frac{v^2}{c^2}}}{1 + \frac{u'_x v}{c^2}}$$

---

### 4. 结论总结

如果你需要从 $S$ 系变换到 $S'$ 系（即已知地面的速度，求飞船内的相对速度），只需将上述公式中的 $v$ 替换为 $-v$，并将带撇和不带撇的符号互换：

- **$x$方向：** $u'_x = \frac{u_x - v}{1 - \frac{u_x v}{c^2}}$
    
- **垂直方向：** $u'_y = \frac{u_y}{\gamma (1 - \frac{u_x v}{c^2})}$
    

---

### 💡 深度理解

- **光速不变验证**：如果在 $S'$ 系中有一束光沿 $x$ 轴传播，即 $u'_x = c$。代入公式：
    
    $$u_x = \frac{c + v}{1 + \frac{cv}{c^2}} = \frac{c + v}{\frac{c + v}{c}} = c$$
    
    这完美证明了无论参考系如何运动，光速始终为 $c$。
    
- **低速退化**：当 $v \ll c$ 时，分母 $1 + \frac{u_x v}{c^2} \approx 1$，公式退化回经典力学的伽利略速度叠加 $u = u' + v$。
