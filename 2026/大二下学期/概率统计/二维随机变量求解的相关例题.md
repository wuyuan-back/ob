这道概率统计题目要求计算二维随机变量函数 $Z$ 的概率密度。我们需要利用卷积公式或分布函数法（累积分布函数法）来求解。

题目给出的联合概率密度函数为：

$$f(x,y) = \begin{cases} x+y, & 0<x<1, 0<y<1 \\ 0, & \text{其他} \end{cases}$$

---

## (1) 求 $Z = X + Y$ 的概率密度 $f_Z(z)$

由于 $X$ 和 $Y$ 都在 $(0, 1)$ 范围内，则 $Z$ 的取值范围是 $(0, 2)$。我们需要分段讨论：

### 情况 1：$0 < z \le 1$

此时积分区域为由 $x=0, y=0, x+y=z$ 围成的三角形。

$$f_Z(z) = \int_0^z f(x, z-x) dx = \int_0^z (x + z - x) dx = \int_0^z z dx = z^2$$

### 情况 2：$1 < z < 2$

此时积分区域受限于 $x<1$ 和 $y<1$。积分限变为从 $z-1$ 到 $1$。

$$f_Z(z) = \int_{z-1}^1 f(x, z-x) dx = \int_{z-1}^1 z dx = z(1 - (z-1)) = z(2-z) = 2z - z^2$$

### 结论

$$f_Z(z) = \begin{cases} z^2, & 0 < z \le 1 \\ 2z - z^2, & 1 < z < 2 \\ 0, & \text{其他} \end{cases}$$

---

## (2) 求 $Z = XY$ 的概率密度 $f_Z(z)$

$Z$ 的取值范围是 $(0, 1)$。我们先求分布函数 $F_Z(z) = P(XY \le z)$，再求导。

### 步骤 1：建立积分表达式

在单位正方形区域内，$XY \le z$ 意味着 $y \le \frac{z}{x}$。

$$F_Z(z) = \iint_{xy \le z} f(x,y) dx dy$$

积分区域分为两部分：

1. $0 < x \le z$ 时，$y$ 的范围是 $(0, 1)$。
    
2. $z < x < 1$ 时，$y$ 的范围是 $(0, \frac{z}{x})$。
    

### 步骤 2：计算分布函数

$$F_Z(z) = \int_0^z \int_0^1 (x+y) dy dx + \int_z^1 \int_0^{z/x} (x+y) dy dx$$

- **左半部分：** $\int_0^z [xy + \frac{1}{2}y^2]_0^1 dx = \int_0^z (x + \frac{1}{2}) dx = [\frac{1}{2}x^2 + \frac{1}{2}x]_0^z = \frac{1}{2}z^2 + \frac{1}{2}z$
    
- **右半部分：** $\int_z^1 [xy + \frac{1}{2}y^2]_0^{z/x} dx = \int_z^1 (z + \frac{z^2}{2x^2}) dx = [zx - \frac{z^2}{2x}]_z^1 = (z - \frac{z^2}{2}) - (z^2 - \frac{z}{2}) = \frac{3}{2}z - \frac{3}{2}z^2$
    

合并得：$F_Z(z) = (\frac{1}{2}z^2 + \frac{1}{2}z) + (\frac{3}{2}z - \frac{3}{2}z^2) = 2z - z^2$

### 步骤 3：求导得出概率密度

$$f_Z(z) = \frac{d}{dz}F_Z(z) = 2 - 2z$$

### 结论

$$f_Z(z) = \begin{cases} 2 - 2z, & 0 < z < 1 \\ 0, & \text{其他} \end{cases}$$
