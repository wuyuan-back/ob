# 🧠 含有运算放大器（Op-Amp）的电阻电路分析

## 1. 基础概念

**运算放大器 (Operational Amplifier)** 是一种具有很高增益的直流耦合放大器件。在理想化分析中，我们通常关注其受控源模型。

### 理想运放的三个“金科玉律”

为了简化电阻电路分析，我们假设运放是**理想的**：

1. **虚短 (Virtual Short):** $u_+ = u_-$ （同相端与反相端电压相等）。
    
2. **虚断 (Virtual Break):** $i_+ = i_- = 0$ （输入阻抗无穷大，不抽取电流）。
    
3. **输出阻抗为零:** 输出端可以提供任意所需的电流。
    

---

## 2. 核心分析方法：结点电压法

分析含有运放的电路时，最有效的方法通常是**结点电压法**。

### 分析步骤：

1. **标定参考点：** 选定公共地（GND）。
    
2. **应用虚短：** 将 $u_+$ 的电压直接赋给 $u_-$。
    
3. **KCL 列方程：** 在 **反相输入端 ($u_-$)** 或 **同相输入端 ($u_+$)** 列出 KCL 方程。
    
    - _注意：通常不要在输出端列 KCL，因为输出电流 $i_o$ 是未知的。_
        
4. **代数求解：** 解方程组得到输出电压 $u_o$ 与输入电压的关系。

## 3. 典型电路模型

### 3.1 反相放大器 (Inverting Amplifier)

![](https://encrypted-tbn0.gstatic.com/licensed-image?q=tbn:ANd9GcQCelc2gwzfu41eeCqwz69tOexx0tOj3EQwhbtoA0ollyMMaYAyQVfmYK_xgYaOdgC8z32rv30Yv9oFgV-VrfEkQ30XmXOL0PhDo-RIdl6oE0WqylI)

- **输入关系：** 信号从反相端输入，同相端接地。
    
- **增益公式：**
    
    $$A_v = \frac{u_o}{u_i} = -\frac{R_f}{R_1}$$
    
- **特点：** 输出与输入反相（相位差 180°）。
    

### 3.2 同相放大器 (Non-inverting Amplifier)

![](https://encrypted-tbn1.gstatic.com/licensed-image?q=tbn:ANd9GcQjiAQC4OEx2jDrxiX6Eq_MJGSNtu66RjJeU2H2zJPtgSPmKJdykUpT2M6bBNISG7D6fra1SK8KEqCcINRHJOmyeLELCRlZBc0LP6b6W4ANjMIhK18)

- **输入关系：** 信号从同相端输入。
    
- **增益公式：**
    
    $$A_v = \frac{u_o}{u_i} = 1 + \frac{R_f}{R_1}$$
    
- **特点：** 输出与输入同相，增益永远 $\ge 1$。
    

### 3.3 电压跟随器 (Voltage Follower)

- **配置：** 将同相放大器的 $R_f$ 设为 0，$R_1$ 设为 $\infty$。
    
- **公式：** $u_o = u_i$
    
- **用途：** 阻抗匹配、信号隔离（高输入阻抗，低输出阻抗）。

## 4. 进阶：组合电路分析

> [!TIP] 叠加定理的应用
> 
> 对于具有多个输入源的线性运放电路（如加法器、减法器），使用**叠加定理**往往比列复杂的 KCL 更快。

### 加法器 (Summing Amplifier)

输出电压是多个输入电压的加权和：

$$u_o = -\left( \frac{R_f}{R_1}u_1 + \frac{R_f}{R_2}u_2 + \dots + \frac{R_f}{R_n}u_n \right)$$

### 差分放大器 (Difference Amplifier)

利用电阻比例实现减法：

若 $\frac{R_f}{R_1} = \frac{R_3}{R_2}$，则：

$$u_o = \frac{R_f}{R_1}(u_2 - u_1)$$

---

## 5. 易错点提醒 ⚠️

- **饱和状态：** 运放的输出不能超过电源电压 ($V_{CC}$ 或 $V_{EE}$)。一旦计算出的 $u_o$ 超过电源轨，运放进入饱和区，不再满足虚短。
    
- **反馈路径：** 只有在**负反馈**（输出通过电阻连回反相端）的情况下，虚短才成立。
    
- **接地：** 始终检查运放的参考地电位。
