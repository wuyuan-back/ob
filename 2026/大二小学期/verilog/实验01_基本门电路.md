# 实验 01：基本门电路与仿真

## 实验目标

- 掌握 Verilog 模块的基本结构
- 学会使用 `assign` 描述组合逻辑
- 掌握 testbench 编写与波形仿真
- 对比门级原语与数据流描述的区别

## 实验原理

### 基本门电路

| 门类型 | 逻辑表达式 | 门级原语 |
|--------|-----------|---------|
| 与门 | `y = a & b` | `and(y, a, b)` |
| 或门 | `y = a \| b` | `or(y, a, b)` |
| 非门 | `y = ~a` | `not(y, a)` |
| 与非门 | `y = ~(a & b)` | `nand(y, a, b)` |
| 或非门 | `y = ~(a \| b)` | `nor(y, a, b)` |
| 异或门 | `y = a ^ b` | `xor(y, a, b)` |

## 代码实现

### 数据流描述（`assign`）

```verilog
module gates (
    input  a, b,
    output y_and, y_or, y_not,
    output y_nand, y_nor, y_xor
);
    assign y_and = a & b;
    assign y_or  = a | b;
    assign y_not = ~a;
    assign y_nand = ~(a & b);
    assign y_nor  = ~(a | b);
    assign y_xor  = a ^ b;
endmodule
```

### 门级原语描述

```verilog
module gates_primitive (
    input  a, b,
    output y_and, y_or, y_not,
    output y_nand, y_nor, y_xor
);
    and  (y_and, a, b);
    or   (y_or,  a, b);
    not  (y_not, a);
    nand (y_nand, a, b);
    nor  (y_nor,  a, b);
    xor  (y_xor,  a, b);
endmodule
```

### Testbench

```verilog
`timescale 1ns / 1ps

module tb_gates;
    reg  a, b;
    wire y_and, y_or, y_not, y_nand, y_nor, y_xor;

    gates uut (
        .a(a), .b(b),
        .y_and(y_and), .y_or(y_or), .y_not(y_not),
        .y_nand(y_nand), .y_nor(y_nor), .y_xor(y_xor)
    );

    initial begin
        $monitor("a=%b b=%b | and=%b or=%b not=%b nand=%b nor=%b xor=%b",
                  a, b, y_and, y_or, y_not, y_nand, y_nor, y_xor);
        a = 0; b = 0; #10;
        a = 0; b = 1; #10;
        a = 1; b = 0; #10;
        a = 1; b = 1; #10;
        $finish;
    end
endmodule
```
