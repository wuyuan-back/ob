# 实验 02：多路选择器与译码器

## 实验目标

- 掌握 `case` / `if-else` 描述组合逻辑
- 实现多路选择器和译码器
- 学会用 LED 验证输出

## 实验原理

### 多路选择器（MUX）

多路选择器根据选择信号从多个输入中选出一个输出。

- 2 选 1：`y = s ? a : b`
- 4 选 1：`case(sel)` 四条分支

### 3-8 译码器

3 位输入 → 8 位独热输出（只有一位为 1）

## 代码实现

### 4 选 1 MUX

```verilog
module mux4 (
    input  [1:0] sel,
    input  [3:0] d0, d1, d2, d3,
    output reg   y
);
    always @(*) begin
        case (sel)
            2'b00:   y = d0;
            2'b01:   y = d1;
            2'b10:   y = d2;
            2'b11:   y = d3;
            default: y = 1'bx;
        endcase
    end
endmodule
```

### 3-8 译码器

```verilog
module decoder_38 (
    input  [2:0] in,
    output reg [7:0] out
);
    always @(*) begin
        case (in)
            3'b000: out = 8'b0000_0001;
            3'b001: out = 8'b0000_0010;
            3'b010: out = 8'b0000_0100;
            3'b011: out = 8'b0000_1000;
            3'b100: out = 8'b0001_0000;
            3'b101: out = 8'b0010_0000;
            3'b110: out = 8'b0100_0000;
            3'b111: out = 8'b1000_0000;
            default: out = 8'b0000_0000;
        endcase
    end
endmodule
```
