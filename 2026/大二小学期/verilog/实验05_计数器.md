# 实验 05：计数器设计

## 实验目标

- 掌握计数器模块的 Verilog 实现
- 实现参数化模 N 计数器
- 理解使能、同步/异步清零

## 代码实现

### 模 10 计数器

```verilog
module counter_10 (
    input            clk, rst_n,
    output reg [3:0] cnt
);
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            cnt <= 4'd0;
        else if (cnt == 4'd9)
            cnt <= 4'd0;
        else
            cnt <= cnt + 1'b1;
    end
endmodule
```

### 参数化模 N 计数器

```verilog
module counter_N #(
    parameter N = 10,
    parameter W = 4
) (
    input            clk, rst_n,
    input            en,
    output reg [W-1:0] cnt,
    output           carry
);
    assign carry = (cnt == N-1) & en;

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            cnt <= 0;
        else if (en) begin
            if (cnt == N-1)
                cnt <= 0;
            else
                cnt <= cnt + 1'b1;
        end
    end
endmodule
```

- [ ] 复位后 cnt = 0
