# 实验 07：流水灯

> 🎉 第一个能上板跑的程序！

## 实验目标

- 综合运用分频器、计数器、移位寄存器
- 实现多种流水灯模式
- 按键切换速度/模式

## 代码实现

```verilog
module led_flash (
    input        clk,
    input        rst_n,
    output reg [7:0] led
);
    reg [23:0] cnt;

    // 分频计数器
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            cnt <= 24'd0;
        else
            cnt <= cnt + 1'b1;
    end

    // 移位寄存器驱动 LED
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n)
            led <= 8'b0000_0001;
        else if (cnt == 24'hffffff)
            led <= {led[6:0], led[7]};  // 循环左移
    end
endmodule
```

## 流水模式扩展

| 模式 | 描述 | 实现方式 |
|------|------|---------|
| 单向 | 从左到右循环 | `{led[6:0], led[7]}` |
| 双向 | 来回扫描 | 增加方向标志位 |
| 跑马灯 | 多灯同时移动 | 拼接多 bit 数据 |
| 呼吸灯 | 渐亮渐暗 | PWM 控制亮度 |

## 检查清单

- [ ] 流水灯每 0.5~1s 移动一次
- [ ] 8 个 LED 全部点亮过
- [ ] 复位后 LED 从第一盏开始
- [ ] 上板运行成功 🎊
