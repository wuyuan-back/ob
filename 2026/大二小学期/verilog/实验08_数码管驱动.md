# 实验 08：数码管动态显示

## 实验目标

- 掌握 7 段数码管译码器设计
- 理解分时复用扫描原理
- 实现 4 位数码管稳定显示

## 原理

### 7 段译码表（共阴极）

```
   aaa
  f   b
  f   b
   ggg
  e   c
  e   c
   ddd
```

### 分时复用

4 位数码管，轮流点亮每一位（~1ms 切换一次），利用视觉暂留看起来全亮。

## 代码实现

### 7 段译码器

```verilog
module seg_decoder (
    input  [3:0] number,
    output reg [6:0] seg
);
    always @(*) begin
        case (number)
            4'h0: seg = 7'b1000000;
            4'h1: seg = 7'b1111001;
            4'h2: seg = 7'b0100100;
            4'h3: seg = 7'b0110000;
            4'h4: seg = 7'b0011001;
            4'h5: seg = 7'b0010010;
            4'h6: seg = 7'b0000010;
            4'h7: seg = 7'b1111000;
            4'h8: seg = 7'b0000000;
            4'h9: seg = 7'b0010000;
            default: seg = 7'b1111111;
        endcase
    end
endmodule
```

### 扫描控制器

```verilog
module seg_scan (
    input            clk, rst_n,
    input  [15:0]    data,           // 4 位 BCD 数据
    output reg [3:0] sel,            // 位选
    output reg [6:0] seg             // 段选
);
    reg [16:0] scan_cnt;
    reg [1:0]  digit;

    always @(posedge clk or negedge rst_n) begin
        if (!rst_n) begin
            scan_cnt <= 0;
            digit    <= 0;
        end else if (scan_cnt == 124_999) begin  // 1ms @50MHz
            scan_cnt <= 0;
            digit    <= digit + 1'b1;
        end else begin
            scan_cnt <= scan_cnt + 1'b1;
        end
    end

    always @(*) begin
        case (digit)
            2'b00: begin sel = 4'b1110; seg = data[3:0];  end
            2'b01: begin sel = 4'b1101; seg = data[7:4];  end
            2'b10: begin sel = 4'b1011; seg = data[11:8]; end
            2'b11: begin sel = 4'b0111; seg = data[15:12];end
        endcase
    end
endmodule
```
