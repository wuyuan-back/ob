# 实验 09：按键消抖与输入检测

## 实验目标

- 理解机械按键的抖动特性
- 掌握同步器和消抖电路设计
- 实现边沿检测

## 原理

机械按键按下/释放时会产生 10-20ms 的抖动（bounce），需要数字滤波消除。

## 代码实现

### 两级同步器 + 边沿检测

```verilog
module debounce (
    input        clk, rst_n,
    input        key_in,
    output reg   key_out,
    output       rising, falling
);
    reg [19:0] cnt;
    reg key_reg1, key_reg2;

    // 两级同步器
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n) begin
            key_reg1 <= 1'b1;
            key_reg2 <= 1'b1;
        end else begin
            key_reg1 <= key_in;
            key_reg2 <= key_reg1;
        end
    end

    // 20ms 延时消抖
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n) begin
            cnt     <= 20'd0;
            key_out <= 1'b1;
        end else begin
            if (key_reg2 != key_out) begin
                if (cnt == 20'd999_999) begin  // 20ms @50MHz
                    cnt     <= 20'd0;
                    key_out <= key_reg2;
                end else begin
                    cnt <= cnt + 1'b1;
                end
            end else begin
                cnt <= 20'd0;
            end
        end
    end

    // 边沿检测
    assign rising  = ~key_out & key_reg2;
    assign falling = key_out & ~key_reg2;
endmodule
```

## 检查清单

- [ ] 两级同步器消除亚稳态
- [ ] 20ms 消抖后按键稳定
- [ ] 上升沿/下降沿检测正确
- [ ] 单次按键只触发一次动作
- [ ] 长按不产生重复触发
