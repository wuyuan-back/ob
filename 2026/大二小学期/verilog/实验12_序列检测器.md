# 实验 12：序列检测器

> 对应课程章节：FSM — 1011 序列检测器（Moore / Mealy）

## 实验目标

- 掌握有限状态机（FSM）设计方法
- 实现 Moore 型与 Mealy 型序列检测器
- 对比两种状态机的输出行为差异

## 原理

- **Moore 型**：输出只取决于当前状态，与输入无关
- **Mealy 型**：输出取决于当前状态和输入，响应更快

检测序列 "1011"：

```
时钟沿:  ↑   ↑   ↑   ↑   ↑   ↑   ↑   ↑
输入:     1   0   1   1   0   1   0   1
Moore:                   ↑               (检测到后下一个时钟沿输出)
Mealy:                   ↑               (检测到当即输出)
```

## 代码实现

### Moore 型

```verilog
module seq_detect_moore (
    input  clk, rst, data_in,
    output seq_detect
);
    parameter Init      = 3'b000,
              Detect_1  = 3'b001,
              Detect_10 = 3'b010,
              Detect_101 = 3'b011,
              Detect_1011 = 3'b100;

    reg [2:0] state;
    assign seq_detect = (state == Detect_1011);

    always @(posedge clk or posedge rst) begin
        if (rst)
            state <= Init;
        else begin
            case (state)
                Init:      state <= data_in ? Detect_1   : Init;
                Detect_1:  state <= data_in ? Detect_1   : Detect_10;
                Detect_10: state <= data_in ? Detect_101 : Init;
                Detect_101:state <= data_in ? Detect_1011: Detect_10;
                Detect_1011:state <= data_in ? Detect_1  : Detect_10;
                default:   state <= Init;
            endcase
        end
    end
endmodule
```

### Mealy 型

```verilog
module seq_detect_mealy (
    input  clk, rst, data_in,
    output seq_detect
);
    parameter Init = 3'b000, Detect_1 = 3'b001,
              Detect_10 = 3'b010, Detect_101 = 3'b011;

    reg [2:0] cstate, nstate;
    assign seq_detect = data_in && (cstate == Detect_101);

    always @(posedge clk or posedge rst) begin
        if (rst) cstate <= Init;
        else     cstate <= nstate;
    end

    always @(*) begin
        case (cstate)
            Init:      nstate = data_in ? Detect_1   : Init;
            Detect_1:  nstate = data_in ? Detect_1   : Detect_10;
            Detect_10: nstate = data_in ? Detect_101 : Init;
            Detect_101:nstate = data_in ? Detect_1   : Detect_10;
            default:   nstate = Init;
        endcase
    end
endmodule
```
