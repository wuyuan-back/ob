# 实验 13：交通灯控制器

## 实验目标

- 用 FSM 实现复杂控制逻辑
- 结合计数器实现定时状态切换
- 数码管倒计时显示

## 原理

### 状态定义

```
            ┌──────────┐
            │  绿灯(G)  │ ← 主干道通行 30s
            └────┬─────┘
                 │ 倒计时到
            ┌────▼─────┐
            │  黄灯(Y)  │ ← 过渡 3s
            └────┬─────┘
                 │ 倒计时到
            ┌────▼─────┐
            │  红灯(R)  │ ← 支干道通行 20s
            └──────────┘
（支干道同理，相位差 180°）
```

## 代码实现

```verilog
module traffic_light (
    input  clk, rst_n,
    output reg [1:0] main_light,    // 00:绿 01:黄 10:红
    output reg [1:0] side_light,
    output reg [5:0] countdown      // 倒计时
);
    parameter GREEN = 2'b00, YELLOW = 2'b01, RED = 2'b10;

    reg [1:0] state, next_state;
    reg [5:0] timer;    // 定时器

    // 状态转移
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n) state <= GREEN;
        else        state <= next_state;
    end

    // 次态逻辑
    always @(*) begin
        case (state)
            GREEN:  next_state = (timer == 0) ? YELLOW : GREEN;
            YELLOW: next_state = (timer == 0) ? RED   : YELLOW;
            RED:    next_state = (timer == 0) ? GREEN : RED;
            default:next_state = GREEN;
        endcase
    end

    // 计数器 + 输出
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n) begin
            timer <= 30;
            main_light <= GREEN;
            side_light <= RED;
            countdown  <= 30;
        end else begin
            if (timer != 0) begin
                timer <= timer - 1;
            end else begin
                case (state)
                    GREEN:  begin timer <= 3;  end
                    YELLOW: begin timer <= 20; end
                    RED:    begin timer <= 30; end
                endcase
            end

            case (state)
                GREEN:  begin main_light <= GREEN; side_light <= RED;   countdown <= timer; end
                YELLOW: begin main_light <= YELLOW;side_light <= RED;   countdown <= timer; end
                RED:    begin main_light <= RED;   side_light <= GREEN; countdown <= timer; end
            endcase
        end
    end
endmodule
```
