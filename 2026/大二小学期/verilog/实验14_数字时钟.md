# 实验 14：数字时钟

## 实验目标

- 综合运用计数器、分频器、数码管
- 实现时/分/秒计时
- 按键校时功能

## 结构

```
clk (50MHz) → 分频器 → 1Hz → 秒计数器(0-59) → 分计数器(0-59) → 时计数器(0-23)
                                 ↓                 ↓                 ↓
                              数码管显示 ←─────────┴─────────────────┘
```

## 代码骨架

```verilog
module digital_clock (
    input  clk, rst_n,
    input  adj_hour, adj_min,     // 校时按键
    output [6:0] seg,             // 段选
    output [5:0] sel              // 位选（6 位数码管）
);
    reg [5:0] hour, min, sec;

    // 秒计数
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n) sec <= 0;
        else if (sec == 59) sec <= 0;
        else                sec <= sec + 1;
    end

    // 分校准（按键按下 +1）
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n) min <= 0;
        else if (adj_min) min <= min + 1;
        else if (min == 59 && sec == 59) min <= 0;
        else if (sec == 59) min <= min + 1;
    end

    // 时校准
    always @(posedge clk or negedge rst_n) begin
        if (!rst_n) hour <= 0;
        else if (adj_hour) hour <= hour + 1;
        else if (min == 59 && sec == 59) begin
            if (hour == 23) hour <= 0;
            else            hour <= hour + 1;
        end
    end

    // 数码管显示（分时复用 + 译码）
    // ...（结合实验 08 的扫描控制）
endmodule
```
