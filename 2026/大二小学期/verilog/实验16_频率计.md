# 实验 16：简易数字频率计

## 实验目标

- 掌握等精度测频原理
- 实现 1Hz ~ 10MHz 范围测量
- 数码管显示测量结果

## 原理

### 等精度测频

在固定闸门时间（1s）内，计数被测信号的脉冲数。

```
频率 = 脉冲数 / 闸门时间
```

## 代码骨架

```verilog
module frequency_counter (
    input  clk,           // 基准时钟 50MHz
    input  rst_n,
    input  sig_in,        // 被测信号
    output reg [31:0] freq_bcd  // BCD 输出
);
    reg [31:0] cnt_sig;   // 信号计数值
    reg [31:0] cnt_ref;   // 基准时钟计数值
    reg gate;             // 闸门信号

    // 1s 闸门生成
    // 闸门内同时计数 sig_in 和 clk
    // 频率 = cnt_sig / cnt_ref * 50MHz

    // ...
endmodule
```
