微分方程
- 可分离变量的微分方程
	- 可化为齐次的方程->$u=\frac{y}{x}$
		- 求出$\frac{du}{dx} = a + b\frac{dy}{dx}$再带回原式
- 一阶线性微分方程
	- $\frac{dy}{dx} +P(x)y = Q(x)$
	- Q(x) = 0 直接分离变量
	- Q(x) != 0 等式两边同×$e^{\int P(x)dx}$->积分因子
- 常见变换的解题思路
	- $\frac{dy}{dx}$->$\frac{dx}{dy}$
	- 中间变量u = x+ y等等
- 高阶方程
	- $y^{(n)} = f(x)$
	- $y'' = f(x,y')$
	- $y'' = f(y,y')$
