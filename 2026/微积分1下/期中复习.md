## 微分方程
- 可分离变量的微分方程
	- 可化为齐次的方程->$u=\frac{y}{x}$
		- 求出$\frac{du}{dx} = a + b\frac{dy}{dx}$再带回原式
- 一阶线性微分方程
	- $\frac{dy}{dx} +P(x)y = Q(x)$
	- Q(x) = 0 直接分离变量
	- Q(x) != 0 等式两边同×$e^{\int P(x)dx}$->积分因子
- 常见变换的解题思路（根据题目尝试）
	- $\frac{dy}{dx}$->$\frac{dx}{dy}$
	- 中间变量u = x+ y,u = y/x,u=x^2
- 高阶方程
	- $y^{(n)} = f(x)$
	- $y'' = f(x,y')$
		- 令p=y',y''=p',高阶转为一阶
	- $y'' = f(y,y')$
		- 依旧令 p = y'但因为式子中不含x所以，$y'' = \frac{dp}{dx}= \frac{dp}{dy} \cdot \frac{dy}{dx}$
- ==常系数齐次线性方程==
	- $y'' + P(x)y' + Q(x)y= 0$
		- 求通解
		- 特征方程$r^2 = Pr+Qr$,解出$\alpha \beta$,$y = Ae^{\alpha x} + Be^{\beta x}$
		- 带回原式求出AB
- ==常系数非齐次线性方程==
	- $y'' + P(x)y' + Q(x)y= f(x)$
		- 求通解f(x) =0
		- 求特解
			- $f(x) = e^{\lambda x}P_m(x)$ m表示为x的m次项式
				- $y^* = x^kQ_m(x)e^{\lambda x}$
					- $\lambda$不为特征根,k=0
					- $\lambda$为单根,k=1
					- $\lambda$ 为重根k =2
				- 代入比较系数求,$Q_m(x)$的系数
			- $f(x) = e^{\lambda x}[P_l(x)cos(wx) + P_n(x)sin(wx)]$
				- $f(x) = x^k e^{\lambda x}[R_max(x)cos(wx) + S_max(x)sin(wx)]$
					-  $\lambda$不为特征根,k=0
					-  $\lambda$为特征根,k=1
				- 特别注意：即使f(x)中只含有$sin 或者cos$,特解中==必须同时包含==$sin和cos$
			- 特别的m=0,多项式为1
	- 题型变式->**特解**－**特解** = **齐次解**，通过特解的分析找通解
题型变化很多，注意结合课后习题和往年题对比研究 ^7a6812
对于积分后的ln函数，若其中的值非负，则绝对值号删去，如果无法判断，则保留绝对值。
- 一阶线性微分方程可以省去
- 其他需要注意
