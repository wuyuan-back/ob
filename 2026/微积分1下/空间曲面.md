# 🗿 二次曲面与坐标面交线 (Quadric Surfaces & Their Traces)

> [!INFO] 预备知识
> 
> **二次曲面**是三维坐标空间中 $x, y, z$ 的二次方程所表示的曲面，其一般形式为：
> 
> $$Ax^2+By^2+Cz^2+Dxy+\cdots=0$$

---

## 1. 椭球面 (Ellipsoid)

### 标准方程

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1 \quad (a, b, c > 0)$$

### 坐标面交线 (Traces)

你的笔记中提到了这一关键点，即通过令一个变量为常数，观察所得的截面曲线：

$$\begin{cases} \frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1 \\ z = A \quad (|A| \le c) \end{cases} \implies \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 - \frac{A^2}{c^2}$$

- **交线形状分析：**
    
    - 当 $|A| < c$ 时，截面是一个**椭圆**。
        
    - 当 $|A| = c$ 时，截面是一个**点**。
        
    - 当 $|A| > c$ 时，**无交线**。

## 2. 抛物面 (Paraboloid)

### 2.1 椭圆抛物面 (Elliptic Paraboloid)

- **标准方程：** $z = \frac{x^2}{2p} + \frac{y^2}{2q}$ $(p, q > 0 \text{，同号})$
    
- **交线分析：**
    
    - 与 $z=A$ ($A>0$) 平面相截：截面为**椭圆**（方程为 $\frac{x^2}{2pA} + \frac{y^2}{2qA} = 1$）。
        
    - 与 $z=A$ ($A<0$) 平面相截：**无交线**。
        
    - 与 $x=A$ 平面相截：截面为**抛物线**。

### 2.2 双曲抛物面 (Hyperbolic Paraboloid)

- **标准方程：** $z = -\frac{x^2}{2p} + \frac{y^2}{2q}$ $(p, q \text{，同号})$
    

> [!TIP] 别名
> 
> 因其形状，它常被称为 **马鞍面**。

- **交线分析：**
    
    - 与 $z=A$ 平面相截：截面为**双曲线**。当 $A>0$ 时，开口沿 $y$ 轴；当 $A<0$ 时，开口沿 $x$ 轴。
        
    - 与 $x=A$ 平面相截：截面为**抛物线**（开口向上）。
        
    - 与 $y=A$ 平面相截：截面为**抛物线**（开口向下）。

## 3. 双曲面 (Hyperboloids)

### 3.1 单叶双曲面 (Hyperboloid of One Sheet)

- **标准方程：** $\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$
    
- **交线分析：**
    
    - 与 $z=A$ 平面相截：截面为**椭圆**（中心孔径随 $z$ 增大而变大）。

### 3.2 双叶双曲面 (Hyperboloid of Two Sheets)

- **标准方程：** $\frac{x^2}{a^2} - \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$
    
- **交线分析：**
    
    - 与 $z=A$ 平面相截：无交线。
        
    - 与 $x=A$ ($|A|>a$) 平面相截：截面为**椭圆**。

## 4. 旋转曲面 (Surfaces of Revolution)

你的笔记清晰地总结了旋转曲面的生成方法：

### 基本概念

由一条平面曲线绕该平面内的一条直线（旋转轴）旋转一周而成。

**口诀：** 绕谁转，谁不动，另一个替换为平方和开根。

### 替换法则图示

### 方程推导

假设 $yOz$ 平面内的母线为 $f(y, z) = 0$。

- **绕 $z$ 轴旋转：** $z$ 不变，将 $y$ 替换为 $\pm\sqrt{x^2+y^2}$。
    
    方程变为：$f(\pm\sqrt{x^2+y^2}, z) = 0$
    
- **绕 $y$ 轴旋转：** $y$ 不变，将 $z$ 替换为 $\pm\sqrt{x^2+z^2}$。
    
    方程变为：$f(y, \pm\sqrt{x^2+z^2}) = 0$

## 5. 柱面 (Cylindrical Surfaces)

你的笔记中对柱面的定义非常精准。

### 柱面的特征

如果在曲面方程中**缺少一个变量**，则该方程表示一个柱面。其母线平行于缺少的变量所对应的坐标轴。

- **例 1：** $F(x, y) = 0 \implies$ 母线平行于 **$z$ 轴**。
    
- **例 2：** $G(x, z) = 0 \implies$ 母线平行于 **$y$ 轴**。
