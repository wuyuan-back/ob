# 3.3.1 直角坐标

- 设解具有乘积形式
	$V(x,y) = X(x)Y(y)$
	求出的是一个特殊解，但通过将特解的叠加,可以构造一个一般解
- 代入二维拉普拉斯方程
	- $f(x) + g(y) = 0$
	- 只有在$f和g$都为常数，才成立
	- 在具体问题中，$C_1$为正$C_2$为负
- $\frac{d^2X}{dx^2} = k^2X,\frac{d^2Y}{dy^2} = -k^2Y$
	- $X =Ae^{kx}+Be^{-kx},Y(y) = Csin(ky) +Dcos(ky)$
- 通过边界条件求出常数
- 把分离变量解叠加起来构造一个通解
	- 通过傅里叶技巧求出系数$C_n$
- 得到一般解
- 用到的特性->$sin(n\pi y/a)$完备，而且具有正交性
