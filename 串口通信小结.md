# 串口通信在MSPM0上的实现
## 对sys进行设置
- 串口的时钟来源有三个：
	1.**BUSCLK**：由内部高频振荡器提供的CPU时钟，通常芯片出厂时设置为了32MHz。
	2.**MFCLK**：只能使用固定的4MHz时钟。开启的话需要配置时钟树的SYSOSC_4M分支，才能够正常开启。
	3.**LFCLK**：由内部的低频振荡器提供时钟（32KHz）。在运行、睡眠、停止和待机模式下有效，使用该时钟可以实现更低的功耗。
	本案例使用的是MFCLK作为串口的时钟来源，开启MFCLK需要在 SYSCONFIG 中配置时钟树。在 SYSCONFIG左侧的选项卡中找到SYSCTL选项，在选项页中找到 Use Clock Tree 进行打勾，以开启时钟树的配置。
