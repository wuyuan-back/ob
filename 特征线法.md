# 📝 数学笔记：二阶常系数 PDE 的算子分解法

## 1. 核心场景

针对**常系数**、**双曲型**二阶偏微分方程的化简：

$$A \frac{\partial^2 z}{\partial x^2} + B \frac{\partial^2 z}{\partial x \partial y} + C \frac{\partial^2 z}{\partial y^2} = 0$$

目标是通过线性代换 $\begin{cases} u = x + m y \\ v = x + n y \end{cases}$ 将其化简为标准型：$\frac{\partial^2 z}{\partial u \partial v} = 0$。

---

## 2. 认知黑客：算子分解路径 (The Shortcut)

无需使用复杂的链式法则展开，直接利用**代数与几何的对等性**。

### Step 1: 算子多项式化

将偏导算子看作变量 $D_x$ 和 $D_y$，写出特征方程并因式分解：

$$A D_x^2 + B D_x D_y + C D_y^2 = (a_1 D_x + b_1 D_y)(a_2 D_x + b_2 D_y)$$

### Step 2: 垂直变换原理 (Orthogonality)

要使混合偏导项以外的项（如 $\frac{\partial^2 z}{\partial u^2}$）消失，新坐标 $u$ 的方向必须与其中一个算子方向**正交**。

- **规律：** 若算子为 $(a D_x + b D_y)$，则对应的代换变量应满足其系数向量 $(1, m)$ 与算子系数向量 $(a, b)$ 的某种特定抵消关系。
    
- **口算准则：** 算子作用在变量上结果为 0。
    
    > $$(a \frac{\partial}{\partial x} + b \frac{\partial}{\partial y})(x + my) = 0 \implies a(1) + b(m) = 0 \implies m = -\frac{a}{b}$$
    

---

## 3. 经典案例解构 (以本题为例)

**原方程：** $6 \frac{\partial^2 z}{\partial x^2} + \frac{\partial^2 z}{\partial x \partial y} - \frac{\partial^2 z}{\partial y^2} = 0$

1. **分解算子：**
    
    $(3 D_x - D_y)(2 D_x + D_y) z = 0$
    
2. **匹配 $u = x - 2y$：**
    
    验证算子 $(2 D_x + D_y)$：$2(1) + 1(-2) = 0$。匹配成功！
    
3. **确定 $v = x + ay$：**
    
    令剩下的算式作用在 $v$ 上等于 0：
    
    $(3 \frac{\partial}{\partial x} - \frac{\partial}{\partial y})(x + ay) = 0$
    
    $3(1) - 1(a) = 0 \implies \mathbf{a = 3}$
    

---

## 4. 背后原理：为什么可以这样做？

- **特征线 (Characteristics)：** 每个一阶算子定义了一个特征方向。
    
- **解构本质：** 这种代换实际上是将坐标轴旋转/拉伸到与方程的“特征线”重合。在特征坐标系下，波动只沿着 $u$ 和 $v$ 传播，因此二阶导数项相互抵消，只剩下反映它们交织关系的混合项。
    

---

## 5. 常见坑点

- [ ] **注意正负号：** 分解算子时，务必检查交叉项系数是否与原方程一致。
    
- [ ] **适用范围：** 仅适用于**常系数**方程。如果系数含有 $x$ 或 $y$，则需要解特征方程微分方程。
