# 📝 信号与系统：零状态响应 (Zero-State Response) 全攻略

## 1. 核心定义

**零状态响应 $r_{zs}(t)$** 指系统在**初始状态为零**（$r^{(k)}(0_-) = 0$）时，仅由外部激励 $e(t)$ 引起的响应。

- **数学表达**：$r_{zs}(t) = e(t) * h(t)$
    
- **特征**：它包含了系统的**暂态分量**（自由响应的一部分）和**稳态分量**（强迫响应）。
    

---

## 2. 求解三大路径

### 路径一：卷积积分法 (Convolution)

这是最符合信号处理直觉的方法。

1. **求冲激响应 $h(t)$**：令输入为 $\delta(t)$。
    
2. **执行卷积**：$r_{zs}(t) = \int_{0}^{t} e(\tau)h(t-\tau)d\tau$。
    

> [!TIP] 适用场景
> 
> 当输入信号 $e(t)$ 比较怪异（如分段函数、非指数函数）时，卷积法最通用。

### 路径二：阶跃/冲激转换法 (Step/Impulse Transformation)

利用 LTI 系统的算子特性：

1. 若 $e(t) = u(t)$，响应即为阶跃响应 $g(t)$。
    
2. 若输入是 $e(t)$ 的组合（如 $e'(t)+3e(t)$），则：
    
    $$r_{zs}(t) = g'(t) + 3g(t)$$
    

> [!TIP] 适用场景
> 
> 当微分方程右端项包含多阶导数时，先求 $g(t)$ 再求导，能有效避免处理高阶 $\delta$ 函数。

### 路径三：经典时域法 (Classical Method)

1. **求特解 $r_p(t)$**：根据激励信号的形式确定。
    
2. **求齐次解 $r_h(t)$**：根据特征根确定。
    
3. **求 $0_+$ 初始值**：利用**状态跳变法**分析从 $0_-$ 到 $0_+$ 的变化。
    
4. **定常数**：代入 $0_+$ 状态解出齐次解系数。

## 3. 单位响应求解要点

### 单位冲激响应 $h(t)$

- **方程形式**：左边 $= \delta(t)$。
    
- **跳变规则**：若左边最高阶为 $n$，则 $h^{(n-1)}(0_+) = 1/a_n$，低阶导数在 $0_+$ 均为 0。
    

### 单位阶跃响应 $g(t)$

- **方程形式**：左边 $= u(t)$。
    
- **跳变规则**：若右边最高阶 $m$ < 左边最高阶 $n$，则 $g(t)$ 及其各阶导数在 $0_+$ 连续（均为 0）。

## 4. 典型例题分析

### 例题 1：基础阶跃响应与冲激响应

**题目**：已知微分方程 $r'(t) + 2r(t) = e(t)$，求 $g(t)$ 和 $h(t)$。

**解析**：

1. **求 $g(t)$**：令 $e(t)=u(t)$。
    
    - 特解：$r_p = 1/2 = 0.5$。
        
    - 齐次解：$Ce^{-2t}$。
        
    - 初始值：$m=0 < n=1$，状态连续，$g(0_+) = g(0_-) = 0$。
        
    - 定系数：$0 = C + 0.5 \Rightarrow C = -0.5$。
        
    - **结果**：$g(t) = (0.5 - 0.5e^{-2t})u(t)$。
        
2. **求 $h(t)$**：直接对 $g(t)$ 求导。
    
    - $h(t) = \frac{d}{dt}[(0.5 - 0.5e^{-2t})u(t)] = (0 + e^{-2t})u(t) + [g(0_+)]\delta(t)$。
        
    - 因为 $g(0_+)=0$，所以 **$h(t) = e^{-2t}u(t)$**。
        

---

### 例题 2：右端带导数的复杂情况

**题目**：$r''(t) + 3r'(t) + 2r(t) = e'(t) + 3e(t)$，输入 $e(t) = u(t)$，求零状态响应。

**解析**：

1. **寻找基础响应 $g_0(t)$**：先看左边 $= u(t)$ 的情况。
    
    - 特征根 $\lambda = -1, -2$。
        
    - 解得 $g_0(t) = (0.5 - e^{-t} + 0.5e^{-2t})u(t)$。
        
2. **利用算子叠加**：由于右边是 $e' + 3e$，利用线性性质：
    
    $$r_{zs}(t) = g_0'(t) + 3g_0(t)$$
    
3. **计算**：
    
    - $g_0'(t) = (e^{-t} - e^{-2t})u(t)$ （此处 $g_0(0_+)=0$ 无 $\delta$）
        
    - $3g_0(t) = (1.5 - 3e^{-t} + 1.5e^{-2t})u(t)$
        
4. **结果**：
    
    **$r_{zs}(t) = (1.5 - 2e^{-t} + 0.5e^{-2t})u(t)$**。

## 5. 总结图表：分量辨析

|**响应分量**|**来源**|**包含的内容**|
|---|---|---|
|**自由响应**|齐次解形式 ($Ce^{\lambda t}$)|全部零输入响应 + 零状态响应中的暂态部分|
|**强迫响应**|特解形式|零状态响应中的稳态部分|
|**零状态响应**|全部激励 $e(t)$|部分自由响应 + 全部强迫响应|
