# 📖 电磁学深度笔记：能量分析与恒定电场

## 1. 静电场能量分析：功与能的博弈
在分析导体系统（如电容器）受到静电力 $f_\xi$ 作用发生微位移 $d\xi$ 时，必须根据边界条件锁定能量流向。

### A. 电位不变系统 (Connected to Source)
* **物理过程**：导体移动 $\to$ 电容 $C$ 改变 $\to$ 电源注入电荷 $dQ$ 以维持电压 $\Phi$。
* **能量收支**：
    * **电源做功**：$dW_{source} = \Phi dQ = \Phi^2 dC$
    * **场能变化**：$dW_e = d(\frac{1}{2}C\Phi^2) = \frac{1}{2}\Phi^2 dC$
    * **机械功输出**：$dW_{mech} = dW_{source} - dW_e = \frac{1}{2}\Phi^2 dC$
* **核心推论**：$dW_{mech} = dW_e$。即电源提供的能量中，**刚好一半**变成了场能，另一半输出了功。


### B. 电荷不变系统 (Isolated System)
* **物理过程**：导体移动，无外部电荷补给。
* **能量收支**：$dW_{mech} = -dW_e$。
* **物理直觉**：由于没有外部电源支持，对外做功必须通过**消耗电场自有的能量密度**来实现，导致极板间场强减弱。

## 2. 导电介质中的恒定电场 (Steady Current Field)
恒定电场是“动态的平衡”，其电荷分布恒定，但电荷流速不为零。

### A. 维持机制：这种场是谁产生的？
* **源电荷**：由电源（电动势）维持的**面电荷分布**。在导电率 $\sigma$ 发生变化的地方（如界面或非均匀介质中），会感应出不随时间变化的电荷分布 $\rho_s$。
* **基本方程推导**：
    1. **电流连续性**：$\nabla \cdot \mathbf{J} = 0$ (电荷不堆积)
    2. **欧姆定律**：$\mathbf{J} = \sigma \mathbf{E}$
    3. **电位的引入**：由于 $\nabla \times \mathbf{E} = 0$，可令 $\mathbf{E} = -\nabla \Phi$
    4. **合成方程**：$\nabla \cdot (\sigma \nabla \Phi) = 0$

### B. 若介质均匀 ($\sigma = \text{const}$)
方程简化为 **[[拉普拉斯方程]]**：
$$\nabla^2 \Phi = 0$$
> [!info] 拓展：非均匀介质
> 若 $\sigma$ 随空间位置改变，则 $\nabla \sigma \cdot \nabla \Phi + \sigma \nabla^2 \Phi = 0$。此时电场内部会产生**体分布电荷** $\rho = \mathbf{J} \cdot \nabla(\frac{\varepsilon}{\sigma})$。

## 3. 边界条件的物理本质
界面上的转折反映了不同材料对电流通过的“阻力”差异。

1. **切向分量** ($E_{1t} = E_{2t}$)：基于能量守恒（沿环路做功为零）。意味着电位 $\Phi$ 在界面处具有 **[[连续性]]**。
2. **法向分量** ($J_{1n} = J_{2n}$)：基于质量/电荷守恒。
    * 导出：$\sigma_1 E_{1n} = \sigma_2 E_{2n}$
    * **面电荷效应**：由高斯定律 $\varepsilon_1 E_{1n} - \varepsilon_2 E_{2n} = \rho_s$ 可知，当 $\sigma_1 \neq \sigma_2$ 时，交界面上必然存在 **[[面电荷密度]]**。

## 4. [[静电比拟]] (The Method of Static Analogy)
这是工程计算中最强大的“偷懒”工具。

### A. 对应关系表
| 静电场 (无源区) | 恒定电场 (导电煤质) |
| :--- | :--- |
| 电位移 $D$ | 电流密度 $J$ |
| 介电常数 $\varepsilon$ | 电导率 $\sigma$ |
| 电容 $C$ | 电导 $G = 1/R$ |
| 通量 $\oint D \cdot dS = Q$ | 电流 $\oint J \cdot dS = I$ |

### B. 经典推论：$RC = \frac{\varepsilon}{\sigma}$
* 如果你已知两个导体在真空中的电容为 $C_0 = \varepsilon_0 \cdot F$，那么当它们被置于电导率为 $\sigma$ 的均匀煤质中时，其间电阻 $R$ 为：
  $$R = \frac{\varepsilon}{\sigma C}$$
* **应用**：计算接地电阻、跨步电压等。

## 🔗 知识图谱扩展
* **[[电功率与焦耳定律]]**：恒定电场中消失的场能转化为热能的过程（$P = \int \mathbf{J} \cdot \mathbf{E} \, dV$）。
* **[[弛豫时间]]**：介质从非平衡态恢复到 $\nabla \cdot \mathbf{J} = 0$ 的过程（$\tau = \varepsilon / \sigma$）。
* **[[唯一性定理]]**：为什么只要边界条件确定，恒定电场的解就是唯一的。
